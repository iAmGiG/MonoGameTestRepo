# MonoGame Test Enterprise Game

This repository contains a starter project for a cross-platform desktop game built with [MonoGame](https://www.monogame.net/). It was generated using the official MonoGame templates for .NET 8.

## Requirements

- [.NET 8 SDK](https://dotnet.microsoft.com/) is required to build and run the project.
- MonoGame templates are already included via NuGet when restoring the project.

## Building and Running

```bash
# Restore dependencies
cd Game
 dotnet restore

# Build and run the game
 dotnet run
```

The default project launches a window displaying the MonoGame logo. Customize `Game1.cs` and add assets in the `Content` folder to start creating your game.
